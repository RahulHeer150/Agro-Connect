# EditProfile Component - Implementation Guide

## Overview
The EditProfile component allows users to edit their profile information.

## ✨ Features Implemented

- ✅ Pre-fills form with existing user data
- ✅ Farmer-specific fields:
  - Farm name, state, district, village
  - Farm size and other details
- ✅ Buyer-specific fields
- ✅ PUT request to update backend
- ✅ Success/error notifications
- ✅ Framer Motion animations

## 🎨 UI/UX Elements
- Profile edit form
- Field pre-filling
- Role-specific fields
- Animations
