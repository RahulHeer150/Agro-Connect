# FarmerLayout Component - Implementation Guide

## Overview
The FarmerLayout component provides a layout wrapper for farmer-specific pages.

## ✨ Features Implemented

- ✅ Combines `Sidebar` + main content area
- ✅ Flex layout with `Outlet` for page content
- ✅ Min-height screen coverage
- ✅ Gray background

## 🎨 UI/UX Elements
- Sidebar layout
- Content outlet
- Background styling
