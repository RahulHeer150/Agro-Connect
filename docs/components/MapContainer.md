# MapContainer Component - Implementation Guide

## Overview
The MapContainer component displays an interactive map with farmer markers.

## ✨ Features Implemented

- ✅ Leaflet-based interactive map
- ✅ Displays farmer markers with custom icons
- ✅ Centered on user's location
- ✅ Zoom controls
- ✅ Tile layer from OpenStreetMap
- ✅ 500px height container
- ✅ Error message if location unavailable

## 🎨 UI/UX Elements
- Interactive map
- Farmer markers
- Zoom controls
- Error handling
