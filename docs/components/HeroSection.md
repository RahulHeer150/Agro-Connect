# HeroSection Component - Implementation Guide

## Overview
The HeroSection component is the main landing page hero with call-to-action.

## ✨ Features Implemented

- ✅ Large headline with green accent
- ✅ Tagline and description
- ✅ Two CTA buttons:
  - "Explore Marketplace"
  - "Become a Farmer" (role-aware redirect)
- ✅ Trust indicators (100% Direct Trade, 0 Middlemen, Fresh Local Produce)
- ✅ Auto-rotating image carousel (3.5s interval)
- ✅ Image animations with fade/scale
- ✅ Responsive 2-column grid
- ✅ Green-50 to white gradient background
