# HexJourney Component - Implementation Guide

## Overview
The HexJourney component displays content in hexagonal grid format.

## ✨ Features Implemented

- ✅ Accepts title and items array
- ✅ Hexagonal image containers (CSS clip-path)
- ✅ Label and description per item
- ✅ 1 → 2 → 4 column responsive layout
- ✅ Beige background (#fdf6ec)
- ✅ Used for journey/process sections

## 🎨 UI/UX Elements
- Hexagonal layout
- Image containers
- Responsive grid
