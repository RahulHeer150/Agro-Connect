# ProductCard Component - Implementation Guide

## Overview
The ProductCard component is a reusable card for displaying products.

## ✨ Features Implemented

- ✅ Product image with hover zoom effect
- ✅ Product name and category
- ✅ Price display in ₹
- ✅ "Buy" button with cart integration
- ✅ Validates user login and buyer role
- ✅ Click to navigate to product details
- ✅ Scales on hover

## 🎨 UI/UX Elements
- Product card
- Image and details
- Buy button
- Hover effects
