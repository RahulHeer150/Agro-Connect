# Register Component - Implementation Guide

## Overview
The Register component provides user registration form.

## ✨ Features Implemented

- ✅ Full name, email, phone, and password inputs
- ✅ Role selection (buyer/farmer)
- ✅ Password confirmation validation
- ✅ Terms & Conditions acceptance checkbox
- ✅ Google login integration
- ✅ Role-based navigation after signup
- ✅ Animated form with staggered inputs

## 🎨 UI/UX Elements
- Registration form
- Role selection
- Validation
- Animations
