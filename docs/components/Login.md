# Login Component - Implementation Guide

## Overview
The Login component provides user login interface.

## ✨ Features Implemented

- ✅ Email and password input fields
- ✅ Backend authentication via API
- ✅ Role-based redirect (farmers → `/farmer-dashboard`, buyers → `/marketplace`)
- ✅ Error message display
- ✅ Loading state during submission
- ✅ Link to registration page

## 🎨 UI/UX Elements
- Login form
- Input fields
- Error handling
- Loading states
