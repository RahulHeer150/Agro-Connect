# PaymentFailure Component - Implementation Guide

## Overview
The PaymentFailure component shows failure page after failed payments.

## ✨ Features Implemented

- ✅ Red themed error screen
- ✅ XCircle icon with rotation animation
- ✅ Reassuring error message
- ✅ Two action buttons:
  - "Retry Payment" → `/checkout`
  - "Back to Cart" → `/cart`
- ✅ Framer Motion animations

## 🎨 UI/UX Elements
- Error screen
- Action buttons
- Animations
