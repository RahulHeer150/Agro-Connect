# Sidebar Component - Implementation Guide

## Overview
The Sidebar component provides side navigation for farmer panel.

## ✨ Features Implemented

- ✅ Links to:
  - Dashboard
  - My Products
  - Add Product
  - Orders
  - Earnings
  - Profile
- ✅ Active link highlighting (green background)
- ✅ Logout button at bottom
- ✅ Uses `NavLink` for active state
- ✅ Fixed width (w-64)

## 🎨 UI/UX Elements
- Navigation links
- Active states
- Logout button
